# Loan-Defaulter-Analysis
## Objective
Finding the who are More Defaulter and Why
## Dataset Overview
- DataSet has taken from the Kaagle
- Number of loan records analyzed: 30000
- Key columns: loan_amount, annual_income, loan_to_income_ratio, housing_type, status (defaulted/paid)
## Key Findings
- Majority of defaulters (51%) live in rented houses and Homeowners have a lower default rate.
- Loans with loan-to-income ratio ≥ 40% in Rent Group have a 100% default rate in the dataset.
## Conclusion
- Renters with high loan-to-income ratios are high-risk borrowers.
- Loan-to-income ratio can be used as a key metric for risk assessment.
